The Torchbearer Project

Torchbearer is a set of libraries for PyTorch designed to aid deep learning research. The base package, torchbearer, is a model fitting library with a series of callbacks and metrics which support advanced visualisations and techniques. The two additional packages, visual and variational provide utilities for visualising convolutional networks and implementing variational auto-encoders.
